# Excel Sales Dashboard

This project is an interactive **Sales Dashboard built in Microsoft Excel** using formulas, charts, and VBA macros.

Instead of PivotTables, the dashboard uses **Excel's built-in functions and dynamic formulas** to calculate and update data based on user selections.

## Features

- KPI Cards for sales metrics
- Sales map visualization
- Gauge chart with rotating needle (VBA)
- Brand comparison chart
- Interactive filtering
- Navigation buttons using VBA macros

## Excel Functions Used

This dashboard is mainly powered by Excel formulas such as:

- SUMIFS (ÇOKETOPLA)
- FILTER (FİLTRE)
- UNIQUE (BENZERSİZ)
- IF
- Logical conditions
- Dynamic ranges

These formulas allow the dashboard to update dynamically when selections change.

## VBA Usage

VBA macros are used for:

- Navigation buttons
- Gauge needle rotation
- Interactive dashboard behavior
